Maintenance

Irreversible actions — a confirmation is required before each operation.

🗑️ Clear All Sales History
Permanently deletes all completed order records from the history log. Active orders in the kitchen are not affected.
🍳 Clear KDS Data
Resets all kitchen item statuses back to Pending. Table assignments and order items are kept — only the cooking progress is cleared.
🪑 Clear Table Data
Removes all active table assignments and their orders. Tables will appear as free/empty. Use this to reset all tables at the end of a session.
📋 Clear Order Table
Clears all records shown on the Active Orders screen. Resets the order list to empty without affecting sales history or kitchen display settings.
💳 Clear Payment Data
Removes saved payment QR settings (Touch & Go and DuitNow URLs). The payment screen will show no QR until new settings are saved.
🪑 Release Table Seating
Select specific occupied tables to release. The selected tables will be freed without affecting other active orders.
